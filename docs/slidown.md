# 技术难点

## 技术难点 

+ 前端技术：JS、CSS、HTML5
+ 后端技术：sinatra、thin
+ Parser：SliMark、Metamark

## 元信息可扩展性

+ 自制**slimark**以解析自定义语法；
+ <kbd>use:plugin</kbd>启用其它插件；
+ <kbd>use:mathjax</kbd>启用**MathJax**支持书写公式；

## 实验性功能

+ 即时的投票
+ PPT遥控器的支持 
+ <kbd>follow.js</kbd>聆听者/演讲者即时交互
+ 可嵌入Web元素：**Flash**、**Video**

## 可能添加特性
